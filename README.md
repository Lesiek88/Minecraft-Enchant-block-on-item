## Minecraft Anvil Mace Blocker

A simple Minecraft plugin that prevents players from modifying a mace item in an anvil.

## Tech Stack

- Java (Bukkit / Spigot API)

## What I Learned

In this project, I learned how to work with Minecraft server events and how to handle inventory interactions using the Bukkit API.
